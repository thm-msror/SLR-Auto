ATLAS Proxy Helper Setup Guide
================================

This guide will help you set up and run the `get_session.py` script on your local machine. This is necessary to "hand over" your institutional login (UDST/IEEE) to the ATLAS cloud application.

Prerequisites:
--------------
1. Python installed on your computer.
2. A stable internet connection.

Setup Steps:
------------
1. Create a new folder on your computer (e.g., "atlas_helper").
2. Place the `get_session.py` script into this folder.
3. Open your terminal or command prompt in that folder.
4. Install the required dependency (Playwright) by running:
   pip install playwright
5. Install the browser drivers by running:
   playwright install chromium

Running the Script:
-------------------
1. Run the script:
   python get_session.py
2. A real browser window will open.
3. Log in to your library/institutional portal as you normally would.
4. Once you are logged in and seeing your dashboard/library, go back to the terminal and press ENTER.
5. A file named `udst_session.json` will be created in the same folder.

Final Step:
-----------
1. Go back to the ATLAS website.
2. Upload the `udst_session.json` file in Step 7.
3. Click "Start Full Pipeline" to begin downloading and analyzing your papers.
